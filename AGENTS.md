# AGENTS.md

## 概要

- ソリューション: `RetDis.sln`
- `RetDis`: 減色処理用のクラスライブラリ
- `RetDisWin`: `RetDis` を利用する Windows Forms UI

## 重要ルール

- このプロジェクトでは、注釈・コメント・補足説明・ドキュメントを日本語で統一する
- 新しく追加するコードコメント、README、設計メモ、作業メモ、UI 文言も原則として日本語で記述する
- 画像処理ロジックは `RetDis` に置き、UI 固有の処理は `RetDisWin` に置く
- フォームに直接ロジックを書き込みすぎず、再利用可能な処理は `RetDis` 側へ寄せる

## 開発前提

- ビルドと実行は Visual Studio で人間が行う前提
- AI は主にコード編集、整理、補助的な実装支援に使う
- 既存の日本語コメントや文体がある場合は、それに合わせる

## 現在の前提

- まだ初期段階のため、過度な抽象化よりも分かりやすさを優先する
- 仕様が固まるまでは、小さく確認しやすい変更を優先する
