# RetDis

RetDis は、レトロゲーム風の減色画像を作るためのツールです。

このリポジトリは、次の 2 つのプロジェクトで構成されています。

- `RetDis`: 減色処理や画像変換ロジックを持つクラスライブラリ
- `RetDisWin`: `RetDis` を利用する Windows Forms ベースの UI

## 目的

入力画像を、少ない色数で構成されたレトロゲーム風の見た目に変換するツールを作ることが、このプロジェクトの目的です。

想定している主な機能領域は次のとおりです。

- パレット管理
- 減色アルゴリズム
- ディザリング処理
- プレビューと保存を行う UI

## ソリューション構成

```text
RetDis.sln
RetDis/      クラスライブラリ (.NET Framework 4.8)
RetDisWin/   Windows Forms アプリケーション (.NET Framework 4.8)
```

## 開発方針

このプロジェクトは AI を活用して開発しますが、製品そのものは AI 製品ではありません。

ビルドと実行は Visual Studio を使って人間が行う前提です。日常的な開発フローは Visual Studio を中心に進めます。

## 開き方

1. Visual Studio で `RetDis.sln` を開きます。
2. 必要であれば Visual Studio の案内に従って復元処理を行います。
3. Visual Studio からソリューションをビルドします。
4. スタートアッププロジェクトを `RetDisWin` に設定して実行します。

## 各プロジェクトの役割

### RetDis

`RetDis` には、再利用可能な画像処理ロジックを配置します。

例:

- パレット定義
- 減色アルゴリズム
- ディザリング実装
- 画像変換ユーティリティ
- 処理オプションや結果モデル

### RetDisWin

`RetDisWin` には、デスクトップ UI を配置します。

例:

- 画像の読み込みと保存の流れ
- プレビュー画面
- パレットや各種設定の操作 UI
- `RetDis` ライブラリとの接続処理

## 現在の状態

このリポジトリは、まだ初期のひな形段階です。

- `RetDis` はクラスライブラリの初期テンプレート
- `RetDisWin` は Windows Forms の初期テンプレート

## 今後の方針

プロジェクトが大きくなっても、責務の分離は維持します。

- 画像処理の本体ロジックは `RetDis` に置く
- UI 固有の振る舞いは `RetDisWin` に置く
- フォームに直接ロジックを書きすぎず、再利用可能な API を優先する
